#pipreqs检查项目依赖
>功能

可以检查项目所需的全部依赖, 并在项目目录下面生成对应的requirements.txt
(修复了python3.4以上在windows下使用报错的问题.)

>使用方法:

```python
# pipreqs 项目地址
```
